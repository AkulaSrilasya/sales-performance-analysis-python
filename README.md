# Sales Performance Analysis using Python

## Overview
This project analyzes retail sales data to identify business insights and performance trends.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Sales by Category Analysis
- Profit by Region Analysis
- Top Selling Sub-Categories
- Sales Visualization using Bar Chart

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Business Insights
- Data Visualization

## Project Outcome
Built a mini sales dashboard analysis project to understand business performance and present insights visually.
